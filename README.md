# ArduinoMorseCode
Uses an Arduino to input and display Morse Code
 
# Parts Used
  * Arduino Uno
  * Breadboard
  * LCD 16x2
  * Pushbutton
  * Potentiometer
  * 220Î© Resistor
  * Jumper Wires
